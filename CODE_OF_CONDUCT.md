# 행동 강령

[English](./CODE_OF_CONDUCT.en.md)

JD-Sync는 작고 실용적이며 환영받는 프로젝트를 지향합니다.

## 기대하는 행동

- 서로를 존중하고 건설적으로 대화합니다.
- 언어, 경험, 배경 차이가 있을 때 선의를 먼저 가정합니다.
- 논의를 프로젝트와 사용자 문제에 집중합니다.
- 사람을 비난하기보다 실행 가능한 피드백을 제공합니다.

## 허용하지 않는 행동

- 괴롭힘, 위협, 차별적 표현
- 동의 없는 개인정보 공개
- issue, pull request, discussion을 반복적으로 방해하는 행동
- 학대적, 성적, 기타 부적절한 행동

## 집행

관리자는 프로젝트와 기여자를 보호하기 위해 댓글을 수정, 숨김, 삭제하거나 issue를 닫고, pull request를 거절하거나, 사용자를 차단할 수 있습니다.

행동 강령 위반을 신고해야 한다면 관리자에게 비공개로 연락해 주세요. 저장소에서 GitHub private vulnerability reporting이나 비공개 연락 채널을 제공하게 되면 민감한 신고에는 해당 채널을 우선 사용합니다.
