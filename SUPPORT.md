# 지원

[English](./SUPPORT.en.md)

사용 중 문제가 생기면 GitHub issue를 열어 주세요. 아래 정보를 포함하면 확인이 빨라집니다.

- 브라우저 버전
- JD-Sync 확장 프로그램 버전
- 문제가 발생한 자소설 페이지 URL 패턴
- 기대한 동작
- 실제로 발생한 동작
- 가능하다면 확장 프로그램 가이드의 최근 동기화 기록

공개 issue에는 Notion 토큰, 쿠키, 비공개 계정 정보, 개인 식별 정보를 포함하지 마세요.

보안 문제는 공개 issue 대신 [SECURITY.md](./SECURITY.md)를 따라 신고해 주세요.
