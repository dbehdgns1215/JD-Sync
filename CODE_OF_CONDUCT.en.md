# Code of Conduct

[한국어](./CODE_OF_CONDUCT.md)

JD-Sync aims to be a small, practical, and welcoming project.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially across language and experience gaps.
- Keep discussion focused on the project and its users.
- Give actionable feedback instead of personal criticism.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Repeatedly derailing issues, pull requests, or discussions.
- Abusive, sexualized, or otherwise inappropriate conduct.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; reject pull requests; or block users when behavior harms the project or its contributors.

If you need to report a conduct issue, contact the maintainers privately. If this repository enables GitHub private vulnerability reporting or private maintainer contact later, prefer that channel for sensitive reports.
