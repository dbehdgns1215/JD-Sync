# Terms of Use

[한국어](./TERMS.md)

Effective date: 2026-06-26

JD-Sync is a Chrome extension that syncs selected schedule metadata from job postings bookmarked by the user into the user's Notion database.

This document describes the basic terms that apply when using JD-Sync.

## Permitted Use

JD-Sync may be used for:

- Personal recruiting schedule management
- Personal job-search records
- Syncing selected metadata from postings the user directly bookmarked
- Displaying those records in a Notion database and Notion Calendar controlled by the user

## Prohibited Use

JD-Sync may not be used for:

- Bulk collection of job posting data
- Commercial use
- Redistribution of job posting data
- Public sharing
- Providing job posting data to third parties
- Circumventing or violating recruiting sites' terms of service, robots.txt rules, technical access restrictions, or applicable laws
- Infringing the rights of recruiting sites, companies, or other rightful owners

## Sync Scope

JD-Sync syncs selected metadata needed for schedule management. This may include company name, posting title, start date, deadline, posting URL, apply URL, posting ID, and duty information.

JD-Sync does not provide a feature that collects full posting HTML, image files, or attachments, and it does not copy those materials into Notion.

## Content Rights

Rights to job posting text, images, company information, and other content belong to the relevant recruiting sites, companies, or rightful owners.

JD-Sync does not replace the original posting. Users must check the original source link for accurate posting details, application requirements, deadlines, and hiring process information.

## User Responsibility

Users must comply with each recruiting site's terms of service and applicable laws when using JD-Sync.

Users are responsible for protecting their Notion token and Notion database access. If a token is exposed, users should immediately rotate or revoke it in Notion.

## Privacy

JD-Sync's data handling practices are described in the [Privacy Policy](./PRIVACY.en.md).

## Changes

These terms may be updated as JD-Sync features, policies, or legal requirements change. Material changes should be documented in the repository and release notes.

## Contact

Open a GitHub issue for general questions about these terms. Do not include Notion tokens, cookies, private account data, or personally identifying information in public issues.
