# Support

[한국어](./SUPPORT.md)

For usage questions, open a GitHub issue with:

- Browser version
- JD-Sync extension version
- Jasoseol page URL pattern, without private query parameters
- What you expected to happen
- What happened instead
- Recent sync logs from the extension guide page, if available

Do not include Notion tokens, cookies, private account data, or personally identifying information in public issues.

For security issues, follow [SECURITY.md](./SECURITY.md) instead of opening a public issue.
